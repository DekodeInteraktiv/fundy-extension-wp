# Donations Extension for WordPress

Provides WordPress integrations for displaying donation forms.

## Repositories

The donations platform is has many components in different repositories:

- [donations-core](https://github.com/DekodeInteraktiv/donations-core)
- [donations-interface](https://github.com/DekodeInteraktiv/donations-interface)
- [donations-form-renderer](https://github.com/DekodeInteraktiv/donations-form-renderer)
- [donations-extension-wp](https://github.com/DekodeInteraktiv/donations-extension-wp)

## Setup

You can get up and running with a few steps:

```bash
npm ci
npm run wp-now
```
